# Mia Carrillo | Front-End Web Developer Portfolio

## Project Overview
A responsive personal portfolio website built from scratch using HTML, CSS, and Bootstrap. Designed as a one-page scrollable experience with smooth navigation, interactive animations, and a clean tech-inspired aesthetic. Built as part of a front-end web development internship at U.S. Hunger.

- Navbar (Bootstrap, fixed-top, hamburger menu on mobile)
- Hero
- About (with interactive CSS timeline)

- Contact (form with accessible labels)
- Footer

## Layout Notes
Flexbox was used for the Bootstrap navbar layout and the footer. Bootstrap's grid system (row/col) was used for the about section two-column layout. CSS Grid was used for the projects section with 3 columns collapsing to 1 on mobile via a media query at 768px. Bootstrap cards were used for the skills section with custom CSS overrides for colors, borders, and hover effects.

## Bootstrap vs Custom CSS
Bootstrap handled the navbar component, responsive grid in the about section, card structure in the skills section, and utility classes for spacing, alignment, and typography throughout. All visual design, animations, color scheme, and section-specific styling was written in custom CSS.

## What I Learned
Building this site taught me how Flexbox and Grid solve different layout problems, how the CSS box model controls spacing, and how media queries make a site responsive. I learned how Bootstrap utility classes can speed up layout work and when it makes more sense to write custom CSS instead. I also experimented with CSS animations using keyframes, pseudo elements for the timeline dots, CSS variables for a consistent color system, and basic JavaScript for the cursor glow and scroll behavior.
